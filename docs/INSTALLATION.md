# Reddington Installation & Deployment Guide\n\n## Single EXE Installer\n\nReddington is distributed as a single executable installer file that automatically handles all setup tasks.\n\n## System Requirements\n\n### Minimum\n- **OS**: Windows 10 (Build 18363+) or later\n- **RAM**: 4 GB\n- **Disk**: 2 GB free space\n- **Display**: 1280x720 resolution minimum\n- **.NET Runtime**: Embedded in installer\n- **Python**: Embedded in installer\n\n### Recommended\n- **OS**: Windows 11 or later\n- **RAM**: 8+ GB\n- **Disk**: 4+ GB SSD\n- **GPU**: NVIDIA/AMD for AI acceleration (optional)\n- **Display**: 1920x1080 or higher\n- **Microphone**: Quality USB microphone\n- **Camera**: HD webcam (1080p+)\n\n## Installation Steps\n\n### For End Users\n\n1. **Download**\n   ```\n   Download: Reddington-Installer.exe\n   Size: ~450 MB\n   ```\n\n2. **Run Installer**\n   - Double-click `Reddington-Installer.exe`\n   - Windows SmartScreen may prompt (allow to proceed)\n   - UAC prompt: Click \"Yes\"\n\n3. **Select Installation Path** (optional)\n   - Default: `C:\\Users\\[Username]\\AppData\\Local\\Reddington`\n   - Custom path supported\n\n4. **Select Features**\n   ```\n   ☑ Desktop Shortcut\n   ☑ Pin to Taskbar\n   ☑ Add to Start Menu\n   ☑ Auto-start on login\n   ```\n\n5. **Wait for Installation**\n   - Extracts files: 30-60 seconds\n   - Downloads dependencies: 1-5 minutes (depends on internet)\n   - Installs Python backend: 30-60 seconds\n   - Total: 2-7 minutes\n\n6. **Completion**\n   - Installer shows \"Installation Complete\"\n   - Option to restart computer\n   - Option to \"Launch Reddington Now\"\n\n### For Developers (Source Build)\n\n1. **Clone Repository**\n   ```bash\n   git clone https://github.com/ByfidTechnologies/Reddington-AI-Assistant.git\n   cd Reddington-AI-Assistant\n   ```\n\n2. **Setup C# Project**\n   ```bash\n   cd src\n   dotnet restore\n   dotnet build\n   ```\n\n3. **Setup Python Backend**\n   ```bash\n   cd backend\n   python -m venv venv\n   .\\venv\\Scripts\\activate  # Windows\n   pip install -r requirements.txt\n   ```\n\n4. **Configure Environment**\n   ```bash\n   copy .env.example .env\n   # Edit .env with your API keys\n   ```\n\n5. **Run Locally**\n   ```bash\n   # Terminal 1: Python Backend\n   cd backend\n   python main.py\n   \n   # Terminal 2: C# Desktop App\n   cd src/Reddington.Desktop\n   dotnet run\n   ```\n\n## First Start Protocol\n\nAfter installation, Reddington automatically launches the **First Start Protocol**:\n\n### Phase 1: Permissions (2-3 min)\n```\n1. Admin Rights Check\n   ↓\n2. Permission Requests\n   - Microphone: Yes/No\n   - Camera: Yes/No\n   - File Access: Yes/No\n   - Admin Rights: Yes/No\n   ↓\n3. System Integration\n   - Set Autostart\n   - Create Shortcuts\n```\n\n### Phase 2: Connectivity (1-2 min)\n```\n1. Internet Test\n   - Download speed: _____ Mbps\n   - Upload speed: _____ Mbps\n   ↓\n2. Service Checks\n   - ChatGPT: 🟢 Connected\n   - Google: 🟢 Connected\n   - Weather: 🟢 Connected\n   - Etc.\n```\n\n### Phase 3: Authentication (5-10 min)\n```\n1. Auto-opens Browser\n2. User logs into services:\n   - Google Account\n   - ChatGPT/OpenAI\n   - Alexa\n   - And more...\n3. System stores auth tokens\n```\n\n### Phase 4: Personalization (3-5 min)\n```\nSystem asks:\n- Your name?\n- Your nickname?\n- Your interests?\n- Your daily routine?\n- Home location?\n- Smart home devices?\n```\n\n### Phase 5: Environment Scanning (2-3 min)\n```\n1. System requests camera access\n2. Scans environment using:\n   - Laptop camera\n   - Connected cameras (if any)\n3. Creates 3D model of room\n4. Saves spatial data locally\n```\n\n### Phase 6: Finalization (1 min)\n```\n1. Setup complete!\n2. Transition to normal Boot Protocol\n3. Reddington ready for use\n```\n\n**Total First Start Time: 15-30 minutes** (first time only)\n\n## Building the Installer\n\n### For Developers\n\n1. **Build Release Binaries**\n   ```bash\n   # C# Desktop\n   dotnet publish -c Release -f net8.0-windows\n   \n   # Python Backend (PyInstaller)\n   cd backend\n   pyinstaller --onefile main.py\n   ```\n\n2. **Create Installer Package**\n   ```bash\n   # Using Inno Setup (Windows)\n   iscc setup.iss\n   \n   # Or using WiX Toolset (advanced)\n   # candle setup.wxs\n   # light setup.wixobj\n   ```\n\n3. **Code Signing** (Optional but recommended)\n   ```bash\n   signtool sign /f certificate.pfx /p password /t http://timestamp.digicert.com Reddington-Installer.exe\n   ```\n\n4. **Output**\n   - `Reddington-Installer.exe` (~450-500 MB)\n   - Ready to distribute\n\n## Configuration Files\n\n### Installation Directory Structure\n\n```\nC:\\Users\\[User]\\AppData\\Local\\Reddington\\\n├── Reddington.exe                    # Main application\n├── .NET Runtime files                # .NET 8 runtime\n├── python/                           # Embedded Python 3.11\n├── backend/                          # Python backend\n├── config/\n│   ├── config.json                   # App configuration\n│   ├── auth.json                     # Encrypted credentials\n│   └── settings.ini                  # User preferences\n├── data/\n│   ├── reddington.db                 # Local database\n│   ├── environment_model.3d          # 3D room model\n│   └── user_profile.json             # User data\n├── logs/\n│   └── reddington.log                # Error logs\n└── cache/                            # Temporary cache\n```\n\n## Post-Installation\n\n### First Launch\n\n1. System checks:\n   - .NET Runtime ✓\n   - Python Backend ✓\n   - Database ✓\n   - Permissions ✓\n\n2. Launches Boot Protocol (see [ARCHITECTURE.md](ARCHITECTURE.md))\n\n3. Shows Dashboard when ready\n\n### Autostart Configuration\n\n**Windows Registry:**\n```\nHKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Run\n  Name: Reddington\n  Value: C:\\Users\\[User]\\AppData\\Local\\Reddington\\Reddington.exe\n  Type: REG_SZ\n```\n\n**Autostart Disable (User Settings)**\n- Settings → General → Auto-start\n- Uncheck \"Launch on system startup\"\n\n## Uninstallation\n\n### Standard Uninstall\n1. Settings → Apps → Apps & features\n2. Search \"Reddington\"\n3. Click \"Uninstall\"\n4. Follow prompts\n5. Reboot (if prompted)\n\n### Manual Cleanup\n```bash\n# Remove installation directory\nrmdir /s /q \"C:\\Users\\[User]\\AppData\\Local\\Reddington\"\n\n# Remove registry entries\nreg delete HKEY_CURRENT_USER\\Software\\Reddington /f\nreg delete HKEY_CURRENT_USER\\Software\\Microsoft\\Windows\\CurrentVersion\\Run /v Reddington /f\n\n# Remove shortcuts\ndel \"%USERPROFILE%\\Desktop\\Reddington.lnk\"\ndel \"%USERPROFILE%\\AppData\\Roaming\\Microsoft\\Windows\\Start Menu\\Programs\\Reddington.lnk\"\n```\n\n## Troubleshooting\n\n### Installation Fails\n\n**Error: \"Windows Defender SmartScreen blocked installation\"**\n- Click \"More info\" → \"Run anyway\"\n- Or disable SmartScreen temporarily\n\n**Error: \"Insufficient disk space\"**\n- Free up 2 GB minimum\n- Or choose custom installation path\n\n**Error: \".NET Runtime installation failed\"**\n- Download .NET 8 runtime separately\n- Run: `dotnet --version`\n- Reinstall Reddington\n\n### Runtime Issues\n\n**Application won't start**\n- Check logs: `%APPDATA%\\Local\\Reddington\\logs\\reddington.log`\n- Clear cache: Delete `%APPDATA%\\Local\\Reddington\\cache\\`\n- Reinstall\n\n**Permissions not working**\n- Run as Administrator\n- Check Windows 11 Settings → Privacy & Security\n\n**Python backend not connecting**\n- Check if port 5000 is available\n- Firewall may be blocking: Allow in Windows Defender\n- Check logs for errors\n\n## Updates\n\n### Automatic Updates\n\nReddington checks for updates on startup:\n- If update available: \"New version ready to install\"\n- Auto-download in background\n- Install on next launch (or manually)\n\n### Manual Updates\n\n```bash\n# Settings → About → Check for Updates\n# Or download new installer and run (replaces old version)\n```\n\n### Update Rollback\n\nIf update causes issues:\n1. Uninstall current version\n2. Download previous version\n3. Reinstall\n\n## Security\n\n### Installer Security\n- **Code Signing**: Signed with trusted certificate\n- **Verification**: SHA-256 hash provided\n- **Transparency**: Source code available on GitHub\n\n### Data Security\n- **Local Storage**: Encrypted with AES-256\n- **No Cloud Sync** (unless user configures)\n- **No Telemetry** (by default)\n\n### User Privacy\n- No data collection without consent\n- All settings editable in config\n- Option to disable: Audio logging, Activity tracking\n\n## Support\n\n**Common Issues**: https://github.com/ByfidTechnologies/Reddington-AI-Assistant/wiki/Troubleshooting\n\n**Report Bugs**: https://github.com/ByfidTechnologies/Reddington-AI-Assistant/issues\n\n**Contact Support**: support@byfidtechnologies.com\n\n---\n\nFor system architecture details, see [ARCHITECTURE.md](ARCHITECTURE.md)\n
